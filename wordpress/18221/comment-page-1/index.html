由liar-soft的新作，嗯，画风还是那么独特，他们的其他作品比如《屋顶上的百合灵》，作画风格都是相当的精致，不是那种千篇一律的gal感觉。这次的游戏主要内容是讲，男主的老婆和别的男人私奔了，于是就只能诱拐小萝莉，于是把一大堆的小萝莉骗到手，最后和萝莉结婚生小孩happy end。


男主松永久秀
因前妻果心居士把家里的财物全部卷走，并且跟别的汉子私奔跑路，曾经一度丧失活下去的勇气。后与平蜘蛛酱结婚，经弟弟松永长赖的介绍于三好家出仕。


平グモちゃん=平蜘蛛酱
与久秀结婚之前曾出嫁过两次，最终皆因婆媳关系不和而导致离婚。父亲是【绍鴎茄子】女儿据推断应该是【九十九发茄子】


楽ちゃん=黒楽茶碗酱
自小暗恋着久秀，是平蜘蛛酱的妹妹，父亲同样是绍鴎茄子，现在正努力的向着姐姐横刀夺爱中。
