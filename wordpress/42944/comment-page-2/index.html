主人公因为要去距离老家很远的学校上学，然后刚好哥哥的咖啡店就在旁边，不用住宿舍来帮我打工吧，在哥哥的邀请下，主人公来到了哥哥的咖啡店打工，可是，没想到，居然有那么多可爱的店员。。。


剧情介绍：
来自哥哥和女仆前辈H的新人教育！
今年的春天开始在这所学院上学的主人公·成濑慧。
由于和老家离得很远，搬到学校附近的哥哥·伊织家借住。
“来我家帮忙打工就行了。不用住宿舍住我家好了”
于是就在哥哥经营的店里开始为了学费和生活费而打工了。
店里的装饰看上去很新奇，竟然连哥哥也扮成了女仆的样子……
“我没跟你说过吗？我们是男性女仆装咖啡店”
总算是完成了第一天的工作，一回家主人公就开始责备哥哥。
“那就别干了？”被这么说也没法反驳。
“但是……这样下去的话也派不上用场”这么说完哥哥开始偷笑，“没办法……我来好好教育一下可爱的弟弟吧”
哥哥露出恶魔一般的表情说出这些话。
主人公究竟会变得怎么样呢……。

性格安静沉稳但没什么主见,
平时会带着眼镜，做女仆的时候会摘掉.

对人和蔼可亲，和主人公关系也很不错，总是洋溢着灿烂的笑容

 

 

 


冴凪 司 （さえなぎ つかさ）　CV：野中みかん

有轻度的男性恐惧症，对于身为普通男性的主人公抱有一点警戒心。
虽然看上去很难接触，其实是个很会为别人着想的好孩子。

 

 

 


成瀬 伊織 （なるせ いおり）
主人公的哥哥。
这家店的老板兼任点心师，具有令海外屈指的有名西式点心店也不得不服的手艺。
在店里像是温柔的大姐姐，但是在家里却会毫不在意地穿着女士内衣到处跑。
平时给人的印象非常可靠，但是内心却很腹黑，最喜欢以欺负弟弟为乐。
就是个抖S的性格。
