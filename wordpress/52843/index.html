触手（学名：tentacle）系存在于多数低等动物身体前端或口周围等处，能自由伸屈之突起物的总称。是一种生物体上的器官或称触须、触角，常见于软体动物，通常是复数，从数根到无法计量之数目的蠕动柔软细长器官。大多用作传感外界环境变化，但触手也可用来攫取物体。
以上摘自百度百科= =
说起触手，就会想到产卵，而说到产卵，就会想到VB系列游戏= =，触手控应该都认识这个系列吧= =
而里番本子里触手范围就比较广些，并不单单是指生物上的触手= =还有各种机器类无生命的触手= =
喜欢触手同时是loli控的推荐3D猫拳系列游戏，各种3Dloli各类触手PLAY= =

资源最难分类整理的就是伪娘，扶她和百合的分类。因为这3类你不把每本汉化本全部看完就很难猜出这本是属于哪类，比如封面有时是萌萌的两个男（女）孩子，到了本子里吃了什么药水&施了什么魔法后，就变成女孩（扶她）了- =

另外最容易分类的就是loli和全彩本- =

以后我还会继续整理这些本子分类，下一弹大概就是最新整理的loli汉化合集了，再之后就是足控和其他分类= =

合集里包括制服触手全6话。
