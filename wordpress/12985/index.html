（推荐向）《时光之刃》

发表于 2012 年 4 月 26 日 由 D.ckv

时光之刃是由Gaijin Entertainmnet制作、Konami发行的动作冒险类游戏，该游戏于2012年4月18日发行，其主角是Ayumi，为一位使用枪剑作战的财宝猎人。在本作中，Ayumi将继续她的冒险，但却因为寻宝而来到了一座奇异而危险的小岛上。

【配置要求】
操作系统: Microsoft Windows XP / Vista / 7;
处理器: Intel Pentium 4 2.6 GHz / AMD 3500 +;
内存: 2 GB;
显卡: nVidia GeForce 7300 GS / ATI Radeon x1300, 256 MB;
声卡: compatible with DirectX;
硬盘所需空间: 3 GB

相较于此作的画面精致度，配置要求很平易近人，我的老电脑跑得还算流畅。（不过我一直认为游戏中凡是用到红色的配色都很诡异）

在游戏性上，如果玩过前作《鬼刃》的话会觉得进步很大，前作的切菜瓜式打法和地图上的重复性让人在后期很没劲，熟练操作后更是能秒杀最终boss；而此作一路打下来，场景的切换不错，难度也基本不会变化太大，虽然一些地方考验操作但也算蛮有趣的，本作中的时光回溯技能是个亮点，用好了会很禽兽哟。

个人认为的退步之处：姿势啊姿势！虽然咋一看比起前作的loli要御姐了很多优雅好看了很多，但——是，持枪时的腿部动作是怎样？！抽了吧？！一定是在抽吧？！当时打得笑死我了！

最后，因为是推荐向，所以，重点就在于——当你把游戏语言切换到日语后，你就会听到熟悉的“乌鲁塞！”（大雾）。

手抽勿怪….
