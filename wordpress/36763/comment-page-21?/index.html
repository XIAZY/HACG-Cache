TMA（英语：Total Media Agency）股份有限公司是日本视频制造商。
它所制作的动画及游戏的cosplay作品有一定影响力。

出演了Fate/stay night的男主角“Gooro”的 鸣泽贤一，不仅长相够雷，在戏中时常还会做出令观众雷到外焦里嫩的表情，且也因此获得众多fans而大红大紫，成为了TMA的招牌，在日本的昵称为“Gooro”，国内的fans则称其为“土狼”。

TMA官网： http://www.tma.co.jp/

注意：TMA公司又2种番号，内容是一样的，格式不一样。
AKB 是DVD番号， HITMA 是蓝光碟番号
