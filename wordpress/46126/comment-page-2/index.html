毎日お疲れ様です。

見た目2D的な3DCGアニメーションによる映像作品です。
あの夏に戻って、エッチなお姉さんに癒やされてみませんか？
アラフォー世代に捧げます

「いちゃらぶ」「ベロチュー」「手コキ」「射乳」「パイズリ」
「ディープスロート」「ぶっかけ」「ゴックン」「大量潮吹き」
これらの趣向の方に是非。

●仕様●
WMV形式による動画ファイルになります。（本編約48分）
HD画質（1280x720pix）にてご提供です。
無料作品の予告篇にて動作確認の上、ご検討ください。

●ストーリー●
1日に数本しかバスが来ない田舎に住んでいたボク。
夏休みはいつも秘密基地に遊びに出かけていた。
そんなある日、雨宿りに飛び込んだバス停で綺麗なお姉さんと出会って…
