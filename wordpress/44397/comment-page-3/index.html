说到3D，就说一下我最近玩的游戏，剑灵。这个内测了那么久的游戏，坚持玩到满级，不过实在是让人感觉太失望了。
同样是NCSOFT，剑灵的可玩性比永恒之塔还差。虽然特效和画质是提升了，不过游戏性实在太差了，你练到满级之后就不知道干嘛了，天天刷狗粮，刷日常，

装备大家都一样，属性也一样，你失去了打BOSS爆极品的那种爽快感，每个人的装备都是一模一样的，最狗屎的是武器居然不能改外形。明明永恒之塔都支持。。。

帮派系统，简直就是一坨屎，你完全察觉不到加入门派对您有任何好处，除了多了一个聊天窗口。没有帮会战，没有攻城战，没有门派驻地。

PVP系统太弱了，想杀人你必须先穿上阵营服装，不过即使你穿上你还是找不到敌对阵营，这个PK地图居然有10几个线，尼玛杀个人还要换线去找人。一个个线切换过去，你想找个敌对势力是多么的困难。。。

然后就是 PVP战场，老他妈掉线，断线我也就不说了，不过尼玛断线之后居然没有保护时间，必须重新排队，如果你正在下副本，超级火大。

我已经放弃这个游戏了。这游戏不怎么花钱倒是真的。

不过你只要花钱买游戏币，你可以在1级立刻拿起满级的武器，满级的装备。拍卖行买材料升级武器就好了，FB什么的完全不用去（拍卖行是几级才能使用我忘记了）

黑完就撤。。。

这是福利#
